Detailed Description

directory - the directory and directory_record datatypes and their associated functions.

All functions, except those with a return value of void, return zero on success and non-zero on failure. A directory record. It's perhaps easier to think of this as a file. It is worth noting, however, that we don't differentiate betewen files and subdirectories.
